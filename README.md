# 💻 DynamicHTML - Interactive Frontend Web Application

## 📌 Project Overview
**DynamicHTML** is a client-side web application developed as part of a standardized academic curriculum challenge during our first year of university (L1 - 2025). The goal of the project was to build a robust, responsive, and fully synchronized multipage platform using foundational frontend technologies and specialized interactive modules.

The application focuses on DOM manipulation, dynamic mathematical calculations, real-time data visualization, and fluid UI design without relying on any backend infrastructure.

## 🚀 Key Modules & Features
The platform consists of 5 dedicated interactive pages:
* **Dashboard / Portal (`index.html`):** A clean entry point introducing the core objective of the web application.
* **Cramer's Solver (`Cramer.html`):** A mathematical tool that dynamically computes 3x3 linear equation systems, rendering full matrices and determinants in real time.
* **Image Resizer (`Image.html`):** A multimedia control interface allowing users to adjust image dimensions on the fly via intuitive range sliders.
* **Automated Ad Banner (`PanneauxPub.html`):** An automated image carousel featuring smooth transitions, hover-to-pause behaviors, and manual control overrides.
* **Data Table & Analytics (`Datatable.html`):** An inventory management layout powered by **DataTables** for sorting and filtering hardware equipment, coupled with **Chart.js** data rendering (Bar and Pie charts) for statistical pricing evaluation.

## 🛠️ Technical Stack
* **Structure & Layout:** HTML5, CSS3 (Custom Flexbox Grid systems)
* **CSS Framework:** Bootstrap 5
* **JavaScript Libraries:** * **jQuery 3.7.1** (DOM manipulation & event handling)
  * **DataTables** (Advanced dynamic client-side tables)
  * **Chart.js v4** (Asynchronous data visualization charts)
* **Iconography:** FontAwesome v6

## ⚙️ How to Run Locally
Since this application runs entirely client-side, no local server installation is required.
1. Clone this repository:
  ```bash
  git clone [https://github.com/aroutiana18/dynamic-html.git](https://github.com/aroutiana18/dynamic-html.git)
  ```
2. Navigate to the project root folder.
3. Open `index.html` directly in any modern web browser (Ubuntu Firefox, Chromium, etc.).
